# Faal-Api 📜 
 A Simple Api For Random Faal

# Installation 💿 
  ```1. git clone https://github.com/pyAref/Faal-Api.git```
  <br/>
  ```2. cd Faal-Api```
  
  npm steps: 
  <br/>
  ``` 1. npm install```
  <br />
  ``` 2. npm start ```
  
# Test 📝  
  ``` 1. npm test ```
 
# Config ⚙️
  you can set port and or host name in json/config.json

Good Luck
